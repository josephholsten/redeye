¶

Recursive scanning method. Simultaneously determines cycles and graph leaves.

 
  scan: (node) ->
    if node in @stack
      @cycles.push @stack[0..-1]
      return
    @stack.push node
    nexts = @inv[node] ? []
    for next in nexts
      @scan next
    unless nexts.length
      unless @state[node] == 'done'
        @loose_ends.push node
    @stack.pop()

Convert the input form of dependencies to a more straightforward version. For instance, it converts

  invert_deps: ->
    for source, targets of @deps
      for target in targets
        sources = (@inv[target] ?= [])
        sources.push source
